TOPOLOGÍAS DE RED

A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con la asignatura de Topologías de red


Topología de red

Es la forma en que se conectan los dispositivos de una red. La topología de una red determina cómo se comunican los dispositivos entre sí y cómo se comparte el tráfico de datos.



LAN

Local Area Network (red de área local). Una LAN es una red de computadoras que conecta computadoras y dispositivos en una ubicación física limitada, como un hogar, una escuela, una oficina o un edificio. Las LAN se utilizan para compartir recursos, como archivos, impresoras y conexiones a Internet.



MAN

Metropolitan Area Network (red de área metropolitana). Una MAN es una red de computadoras que conecta computadoras y dispositivos en una área metropolitana, como una ciudad. Las MAN se utilizan para compartir recursos, como archivos, impresoras y conexiones a Internet, entre ubicaciones distantes dentro de un área metropolitana.



WAN

Wide Area Network (red de área amplia). Una WAN es una red de computadoras que conecta computadoras y dispositivos en una amplia área geográfica, como una ciudad, un país o el mundo. Las WAN se utilizan para compartir recursos, como archivos, aplicaciones y conexiones a Internet, entre ubicaciones distantes.



Normas IEEE

Son un conjunto de estándares desarrollados por el Instituto de Ingenieros Eléctricos y Electrónicos (IEEE). Estos estándares se utilizan en una amplia gama de tecnologías, incluidas las redes informáticas, los sistemas eléctricos y las telecomunicaciones.



DHCP

Protocolo de configuración dinámica de host, es un protocolo de red que se utiliza para asignar automáticamente direcciones IP y otros parámetros de configuración de red a los dispositivos conectados a una red. DHCP permite a los dispositivos obtener una dirección IP válida sin necesidad de que un administrador de red la configure manualmente.



Dirección IP

Es una etiqueta numérica que identifica de manera lógica y jerárquica a una interfaz y se utilizan para identificar de forma única a cada dispositivo en una red. Esto permite que los dispositivos se comuniquen entre sí y que los datos se envíen y reciban correctamente.



Patch panel

Es un dispositivo de red que se utiliza para organizar y administrar los cables de red. Los patch panels se utilizan en una variedad de entornos, incluyendo centros de datos, salas de servidores y oficinas.



Cable de conexion UTP

(Unshielded Twisted Pair o cable de par trenzado sin blindaje) es un tipo de cable de cobre que se utiliza para conectar dispositivos a una red. Los cables UTP están compuestos por cuatro pares de cables de cobre trenzados.



Ethernet

Es una familia de tecnologías de redes de área local (LAN) que utilizan cableado de cobre o fibra óptica para conectar dispositivos a una red. Ethernet es la tecnología de LAN más popular del mundo y se utiliza en una amplia gama de entornos, desde hogares y oficinas hasta centros de datos y redes de campus.

--


Router

Dispositivo de red que conecta dos o más redes informáticas. Los routers se utilizan para enrutar el tráfico de red entre las redes conectadas. Los routers pueden ser utilizados para conectar redes de diferentes tamaños, desde redes domésticas hasta redes empresariales.



Switch

Dispositivo de red que conecta dispositivos a una red. Los switches se utilizan para crear una red de área local (LAN) o para ampliar una LAN existente. Los switches también se pueden utilizar para conectar LANs a redes de área amplia (WANs).



Hub

Es un dispositivo de red que conecta dispositivos a una red. Los hubs funcionan de forma similar a los switches, pero los hubs no dividen una red en segmentos. Todos los dispositivos conectados a un hub están en el mismo segmento de red.



Encapsulación

El proceso de empaquetar los datos en unidades más pequeñas para su transmisión por la red.



Estándares TIA

Son un conjunto de estándares desarrollados por la Asociación de la Industria de Telecomunicaciones (TIA) para el diseño, instalación y mantenimiento de redes de telecomunicaciones. Los estándares TIA se utilizan ampliamente en todo el mundo para garantizar la interoperabilidad de los equipos de red y la calidad de las instalaciones de red.

SISTEMAS OPERATIVOS

A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con la asignatura de Sistemas Operativos


Sistema operativo

Es un software que gestiona los recursos de un sistema informático.



Software

Es un conjunto de programas, instrucciones y datos que permiten que un dispositivo electrónico funcione. Es la parte intangible de un sistema informático, en contraposición al hardware, que es la parte física.



Copias de seguridad

Es una copia de los datos originales que se realiza con el fin de disponer de un medio para recuperarlos en caso de su pérdida.



Panel de control

Es una interfaz gráfica que permite a los usuarios configurar y administrar los aspectos de un sistema, dispositivo o aplicación.



Virus informáticos

Programas maliciosos que pueden dañar o destruir datos.



Memoria RAM

Memoria de acceso aleatorio que almacena datos que se utilizan actualmente.



BIOS

software almacenado en un pequeño chip de memoria en la placa base de una computadora. Es el primer software que se ejecuta cuando se enciende la computadora, y es responsable de inicializar el hardware y cargar el sistema operativo.



Sistema de archivos

La estructura que organiza y administra cómo se almacenan y se acceden a los archivos en un sistema operativo.



Byte

Es una unidad de información que consta de ocho bits. Los bits son los componentes básicos de la información digital y pueden tener dos valores: 0 o 1. Un byte puede representar un solo carácter, como una letra, un número o un símbolo. También puede representar un valor numérico



Kernel

La parte principal de un sistema operativo que se encarga de la gestión de los recursos de la computadora.



Discos

son dispositivos de almacenamiento de datos que utilizan un sistema de grabación magnético para almacenar información. Los datos se almacenan en forma de patrones de magnetización en la superficie del disco.

--


Procesador

Es el cerebro del sistema, justamente procesa todo lo que ocurre en la PC y ejecuta todas las acciones que existen.



Interfaz Gráfica de Usuario (GUI)

Es un entorno visual que permite a los usuarios interactuar con dispositivos y programas mediante iconos y ventanas, simplificando tareas mediante clics y arrastres en lugar de comandos de texto.



Firewall

Es un sistema de seguridad que protege una red o un dispositivo de accesos no autorizados. Funciona como una barrera que controla el tráfico de datos entrante y saliente, permitiendo solo el tráfico permitido.
